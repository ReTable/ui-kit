## Installation

Use the package manager [pnpm](https://pnpm.io) to install `@tabula/use-previous-value`.

```bash
pnpm add @tabula/use-previous-value
```

You can use [npm](https://npmjs.com) or [yarn](https://yarnpkg.com) too.
