# @tabula/use-flag

Allows to use boolean state as flag..

## Installation

Use the package manager [pnpm](https://pnpm.io) to install `@tabula/use-flag`.

```bash
pnpm add @tabula/use-flag
```

You can use [npm](https://npmjs.com) or [yarn](https://yarnpkg.com) too.
