# Component

We provide icons as React components.

## Import

<br />

<Source code={`
import { UiStringIcon } from '@tabula/ui-data-type-icon';
`} language="ts"/>

<br />

## Default

Data types icons has `16x16` size by default.

<Canvas  layout="centered">
  <UiStringIcon />
</Canvas>

## Styling

You can style icons with CSS.

### Color

All data types icons inherit `currentColor` property.

<Canvas layout="centered">
  <div style={{ color: 'blue' }}>
    <UiStringIcon />
  </div>
</Canvas>

### Size with class

You can provide `className` property.

<Canvas layout="centered">
  <div>
    <style>
      {`
        .ui-string-icon {
          width: 32px;
          height: 32px;
        }
      `}
    </style>

    <UiStringIcon className='ui-string-icon' />
  </div>
</Canvas>

### Size with style

You can provide `style` property.

<Canvas layout="centered">
  <UiStringIcon style={{ width: '32px', height: '32px' }} />
</Canvas>

# Image URL

Also, we provide icons as static images.

## Import

<br />

<Source code={`
import { uiStringIconUrl } from "@tabula/ui-data-type-icon";
`} language="ts"/>

<br />

## CSS / Sass

If you use a Vite bundler, you can use icons in your styles like this:

<Source code={`
.awesome-component {
  background-image: url('@tabula/ui-data-type-icon/string.svg');
}
`} language="css"/>
