# @tabula/ui-checkbox

Checkbox control for forms..

## Installation

Use the package manager [pnpm](https://pnpm.io) to install `@tabula/ui-checkbox`.

```bash
pnpm add @tabula/ui-checkbox
```

You can use [npm](https://npmjs.com) or [yarn](https://yarnpkg.com) too.
