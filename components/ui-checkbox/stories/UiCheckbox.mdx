# UiCheckbox

Checkboxes allow users to select multiple items from a list, or to mark one item as selected.

<Canvas of={Stories.Checked} />

## Options

There are a few general properties which supported by component.

### `isChecked`

This property controls boolean value of checkbox.

<Canvas of={Stories.Checked} />
<Canvas of={Stories.Unchecked} />

### `isIndeterminate`

You can use indeterminate state of checkbox.

<Canvas of={Stories.Indeterminate} />

### `isDisabled`

This property allows to disable a control.

<Canvas of={Stories.CheckedDisabled} />
<Canvas of={Stories.UncheckedDisabled} />
<Canvas of={Stories.IndeterminateDisabled} />
