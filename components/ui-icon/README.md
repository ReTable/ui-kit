# @tabula/ui-icon

Sets of various icon sets.

## Installation

Use the package manager [pnpm](https://pnpm.io) to install `@tabula/ui-icon`.

```bash
pnpm add @tabula/ui-icon
```

You can use [npm](https://npmjs.com) or [yarn](https://yarnpkg.com) too.

## Description

Icons are divided into namespaces:

- `dataTypes`
- `separators`

## Usage

```tsx
import { UiArrayIcon } from '@tabukla/ui-icon/dataTypes';

function ComponentWithIcon() {
  return <UiArrayIcon />
}
```
