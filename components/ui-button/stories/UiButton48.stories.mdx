# UiButton48

Buttons allow users to take actions, and make choices. Simple example:

<Canvas>
  <UiButton48 variant="primary">Click me!</UiButton48>
</Canvas>

## Types

The component supports four types: `button`, `a`, `div` and `link`. By default, the `button` type is used.

### Button

By default, the component renders generic button:

<Canvas>
  <UiButton48 variant="primary">Default</UiButton48>
</Canvas>

You can provide `as="button"` explicitly:

<Canvas>
  <UiButton48 as="button" variant="primary">Button</UiButton48>
</Canvas>

### Anchor

You can render `a` element with providing `as="button"`:

<Canvas>
  <UiButton48 as="a" href="#" target="_blank" variant="primary">Anchor</UiButton48>
</Canvas>

### Div

In rare cases, you can render `div` element with providing `as="div"`:

<Canvas>
  <UiButton48 as="div" variant="primary">Div</UiButton48>
</Canvas>

### Link

You can use `as="link"` and provide `component` property to use `react-router`'s `Link` component:

<Canvas>
  <UiButton48 as="link" component={Link} target="_blank" to="#" variant="primary">Link</UiButton48>
</Canvas>

It can be useful, when you want to have `div` element with button appearance. For example, to render it inside
another `button` element.

## Variants

There are 2 appearance variants, which you can use:

<Canvas>
  <UiButton48 variant="primary">Primary</UiButton48>
  <UiButton48 variant="secondary">Secondary</UiButton48>
</Canvas>

## Options

The component supports a few options.

### `isDisabled`

This property allows to disable button:

<Canvas>
  <UiButton48 as="button" isDisabled variant="primary">Button</UiButton48>
  <UiButton48 as="a" href="#" isDisabled variant="primary">Anchor</UiButton48>
  <UiButton48 as="div" isDisabled variant="primary">Div</UiButton48>
  <UiButton48 as="link" component={Link} isDisabled to="#" variant="primary">Link</UiButton48>
</Canvas>

### `isFrozen`

This property allows to disable button, but keep enabled look and feel:

<Canvas>
  <UiButton48 as="button" isFrozen variant="primary">Button</UiButton48>
  <UiButton48 as="a" href="#" isFrozen variant="primary">Anchor</UiButton48>
  <UiButton48 as="div" isFrozen variant="primary">Div</UiButton48>
  <UiButton48 as="link" component={Link} isFrozen to="#" variant="primary">Link</UiButton48>
</Canvas>

### `isDisabled` and `isFrozen`

The `isDisabled` styles has priority over `isFrozen` when they're used together.

<Canvas>
  <UiButton48 as="button" isDisabled isFrozen variant="primary">Button</UiButton48>
  <UiButton48 as="a" isDisabled isFrozen href="#" variant="primary">Anchor</UiButton48>
  <UiButton48 as="div" isDisabled isFrozen variant="primary">Div</UiButton48>
  <UiButton48 as="link" component={Link} isDisabled isFrozen to="#" variant="primary">Link</UiButton48>
</Canvas>

### `icon`

You can provide an icon component:

<Canvas>
  <UiButton48 as="button" icon={AddIcon} variant="primary">Button</UiButton48>
  <UiButton48 as="a" icon={AddIcon} href="#" target="_blank" variant="primary">Anchor</UiButton48>
  <UiButton48 as="div" icon={AddIcon} variant="primary">Div</UiButton48>
  <UiButton48 as="link" icon={AddIcon} component={Link} target="_blank" to="#" variant="primary">Link</UiButton48>
</Canvas>

We recommend use icons with size of `16x16`.

An icon component should support `className` property.

## Attributes

All component types supports relevant HTML attributes, excluding `aria-disabled` and `disabled`. This attributes are
controlled by the component itself.

Also `href` attribute is restricted `link` type.

For example, you can provide `data-*` attribute for analytics purposes:

<Canvas>
  <UiButton48 as="button" data-track-id="button" variant="primary">Button</UiButton48>
  <UiButton48 as="a" data-track-id="anchor" href="#" target="_blank" variant="primary">Anchor</UiButton48>
  <UiButton48 as="div" data-track-id="div" variant="primary">Div</UiButton48>
  <UiButton48 as="link" data-track-id="link" component={Link} target="_blank" to="#" variant="primary">Link</UiButton48>
</Canvas>
