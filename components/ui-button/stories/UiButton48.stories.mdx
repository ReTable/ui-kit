# UiButton48

Buttons allow users to take actions, and make choices. Simple example:

<Canvas>
  <UiButton48 variant="primary">Clickable Button</UiButton48>
</Canvas>

## Demos

The component supports are many options.

### As

The components supports three modes: `button`, `link` and `visual`. By default, the `button` mode is used.

#### Button

By default, the component renders generic button:

<Canvas>
  <UiButton48 variant="primary">Default</UiButton48>
  <UiButton48 as="button"  variant="primary">Button</UiButton48>
</Canvas>

#### Link

You can use button as link providing `as="link"` property, and additionally `href` and optionally `target`:

<Canvas>
  <UiButton48 as="link" href="#" variant="primary">Link</UiButton48>
  <UiButton48 as="link" href="#" target="_blank" variant="primary">Link with target</UiButton48>
</Canvas>

#### Visual

You can render button as `div` element providing `as="visual"` for rare cases when you can't use `button` element:

<Canvas>
  <UiButton48 as="visual" variant="primary">Visual</UiButton48>
</Canvas>

### Variant

There are 2 variants:

<Canvas>
  <UiButton48 variant="primary">Primary</UiButton48>
  <UiButton48 variant="secondary">Secondary</UiButton48>
</Canvas>

### Disabled

All buttons supports `isDisabled` property:

<Canvas>
  <UiButton48 as="button" isDisabled variant="primary">Button</UiButton48>
  <UiButton48 as="link" isDisabled href="#" variant="primary">Link</UiButton48>
  <UiButton48 as="visual" isDisabled variant="primary">Visual</UiButton48>
</Canvas>

### Frozen

Along with `isDisabled` option, the component supports `isFrozen` option. In that case, button will have enabled visual
style, but will not react on user interactions.

<Canvas>
  <UiButton48 as="button" isFrozen variant="primary">Button</UiButton48>
  <UiButton48 as="link" isFrozen href="#" variant="primary">Link</UiButton48>
  <UiButton48 as="visual" isFrozen variant="primary">Visual</UiButton48>
</Canvas>

### Icon

You can provide an icon component:

<Canvas>
  <UiButton48 icon={AddIcon} variant="primary">Button</UiButton48>
  <UiButton48 icon={ChevronRightIcon} variant="secondary">Button</UiButton48>
</Canvas>

We recommend use icons with size of `16x16`.

An icon component should supports `className` property.

### Analytics

Also, the component supports the `trackId` property. It's adds `data-track-id` property to component for analytics
purposes.

<Canvas>
  <UiButton48 as="button" trackId="my-track-id" variant="primary">Tracked</UiButton48>
</Canvas>
