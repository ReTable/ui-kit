# UiButton20

Buttons allow users to take actions, and make choices. Simple example:

<Canvas className={contractBackground}>
  <UiButton20 variant="contract">Click me!</UiButton20>
</Canvas>

## Types

The component supports four types: `button`, `a`, `div` and `link`. By default, the `button` type is used.

### Button

By default, the component renders generic button:

<Canvas className={contractBackground}>
  <UiButton20 variant="contract">Default</UiButton20>
</Canvas>

You can provide `as="button"` explicitly:

<Canvas className={contractBackground}>
  <UiButton20 as="button" variant="contract">Button</UiButton20>
</Canvas>

### Anchor

You can render `a` element with providing `as="button"`:

<Canvas className={contractBackground}>
  <UiButton20 as="a" href="#" target="_blank" variant="contract">Anchor</UiButton20>
</Canvas>

### Div

In rare cases, you can render `div` element with providing `as="div"`:

<Canvas className={contractBackground}>
  <UiButton20 as="div" variant="contract">Div</UiButton20>
</Canvas>

### Link

You can use `as="link"` and provide `component` property to use `react-router`'s `Link` component:

<Canvas className={contractBackground}>
  <UiButton20 as="link" component={Link} target="_blank" to="#" variant="contract">Link</UiButton20>
</Canvas>

It can be useful, when you want to have `div` element with button appearance. For example, to render it inside
another `button` element.

## Variants

There is single appearance variant, which you can use:

<Canvas className={contractBackground}>
  <UiButton20 variant="contract">Contract</UiButton20>
</Canvas>

## Options

The component supports a few options.

### `isDisabled`

This property allows to disable button:

<Canvas className={contractBackground}>
  <UiButton20 as="button" isDisabled variant="contract">Button</UiButton20>
  <UiButton20 as="a" href="#" isDisabled variant="contract">Anchor</UiButton20>
  <UiButton20 as="div" isDisabled variant="contract">Div</UiButton20>
  <UiButton20 as="link" component={Link} isDisabled to="#" variant="contract">Link</UiButton20>
</Canvas>

### `isFrozen`

This property allows to disable button, but keep enabled look and feel:

<Canvas className={contractBackground}>
  <UiButton20 as="button" isFrozen variant="contract">Button</UiButton20>
  <UiButton20 as="a" href="#" isFrozen variant="contract">Anchor</UiButton20>
  <UiButton20 as="div" isFrozen variant="contract">Div</UiButton20>
  <UiButton20 as="link" component={Link} isFrozen to="#" variant="contract">Link</UiButton20>
</Canvas>

### `isDisabled` and `isFrozen`

The `isDisabled` styles has priority over `isFrozen` when they're used together.

<Canvas className={contractBackground}>
  <UiButton20 as="button" isDisabled isFrozen variant="contract">Button</UiButton20>
  <UiButton20 as="a" isDisabled isFrozen href="#" variant="contract">Anchor</UiButton20>
  <UiButton20 as="div" isDisabled isFrozen variant="contract">Div</UiButton20>
  <UiButton20 as="link" component={Link} isDisabled isFrozen to="#" variant="contract">Link</UiButton20>
</Canvas>

### `icon`

You can provide an icon component:

<Canvas className={contractBackground}>
  <UiButton20 as="button" icon={AddIcon} variant="contract">Button</UiButton20>
  <UiButton20 as="a" icon={AddIcon} href="#" target="_blank" variant="contract">Anchor</UiButton20>
  <UiButton20 as="div" icon={AddIcon} variant="contract">Div</UiButton20>
  <UiButton20 as="link" icon={AddIcon} component={Link} target="_blank" to="#" variant="contract">Link</UiButton20>
</Canvas>

We recommend use icons with size of `16x16`.

An icon component should support `className` property.

## Attributes

All component types supports relevant HTML attributes, excluding `aria-disabled` and `disabled`. This attributes are
controlled by the component itself.

Also `href` attribute is restricted `link` type.

For example, you can provide `data-*` attribute for analytics purposes:

<Canvas className={contractBackground}>
  <UiButton20 as="button" data-track-id="button" variant="contract">Button</UiButton20>
  <UiButton20 as="a" data-track-id="anchor" href="#" target="_blank" variant="contract">Anchor</UiButton20>
  <UiButton20 as="div" data-track-id="div" variant="contract">Div</UiButton20>
  <UiButton20 as="link" data-track-id="link" component={Link} target="_blank" to="#" variant="contract">Link</UiButton20>
</Canvas>
