# UiButton20

Buttons allow users to take actions, and make choices. Simple example:

<Canvas className={contractBackground}>
  <UiButton20 variant="contract">Clickable Button</UiButton20>
</Canvas>

## Demos

The component supports are many options.

### As

The components supports three modes: `button`, `link` and `visual`. By default, the `button` mode is used.

#### Button

By default, the component renders generic button:

<Canvas className={contractBackground}>
  <UiButton20 variant="contract">Default</UiButton20>
  <UiButton20 as="button"  variant="contract">Button</UiButton20>
</Canvas>

#### Link

You can use button as link providing `as="link"` property, and additionally `href` and optionally `target`:

<Canvas className={contractBackground}>
  <UiButton20 as="link" href="#" variant="contract">Link</UiButton20>
  <UiButton20 as="link" href="#" target="_blank" variant="contract">Link with target</UiButton20>
</Canvas>

#### Visual

You can render button as `div` element providing `as="visual"` for rare cases when you can't use `button` element:

<Canvas className={contractBackground}>
  <UiButton20 as="visual" variant="contract">Visual</UiButton20>
</Canvas>

### Variant

There is single variant:

<Canvas className={contractBackground}>
  <UiButton20 variant="contract">Contract</UiButton20>
</Canvas>

### Disabled

All buttons supports `isDisabled` property:

<Canvas className={contractBackground}>
  <UiButton20 as="button" isDisabled variant="contract">Button</UiButton20>
  <UiButton20 as="link" isDisabled href="#" variant="contract">Link</UiButton20>
  <UiButton20 as="visual" isDisabled variant="contract">Visual</UiButton20>
</Canvas>

### Frozen

Along with `isDisabled` option, the component supports `isFrozen` option. In that case, button will have enabled visual
style, but will not react on user interactions.

<Canvas className={contractBackground}>
  <UiButton20 as="button" isFrozen variant="contract">Button</UiButton20>
  <UiButton20 as="link" isFrozen href="#" variant="contract">Link</UiButton20>
  <UiButton20 as="visual" isFrozen variant="contract">Visual</UiButton20>
</Canvas>

### Icon

You can provide an icon component:

<Canvas className={contractBackground}>
  <UiButton20 icon={AddIcon} variant="contract">Button</UiButton20>
</Canvas>

We recommend use icons with size of `16x16`.

An icon component should supports `className` property.

### Analytics

Also, the component supports the `trackId` property. It's adds `data-track-id` property to component for analytics
purposes.

<Canvas className={contractBackground}>
  <UiButton20 as="button" trackId="my-track-id" variant="contract">Tracked</UiButton20>
</Canvas>
