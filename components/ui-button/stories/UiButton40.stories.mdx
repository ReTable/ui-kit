# UiButton40

Buttons allow users to take actions, and make choices. Simple example:

<Canvas>
  <UiButton40 variant="primary">Clickable Button</UiButton40>
</Canvas>

## Demos

The component supports are many options.

### Type

The components support three types: `button`, `link` and `visual`. By default, the `button` type is used.

#### Button

By default, the component renders generic button:

<Canvas>
  <UiButton40 variant="primary">Default</UiButton40>
  <UiButton40 type="button" variant="primary">Button</UiButton40>
</Canvas>

#### Link

You can use button as link providing `type="link"` property, and additionally `href` and optionally `target`:

<Canvas>
  <UiButton40 href="#" type="link" variant="primary">Link</UiButton40>
  <UiButton40 href="#" target="_blank" type="link" variant="primary">Link with target</UiButton40>
</Canvas>

#### Visual

You can render button as `div` element providing `type="visual"` for rare cases when you can't use `button` element:

<Canvas>
  <UiButton40 type="visual" variant="primary">Visual</UiButton40>
</Canvas>

### Variant

There are 4 variants:

<Canvas>
  <UiButton40 variant="primary">Primary</UiButton40>
  <UiButton40 variant="secondary">Secondary</UiButton40>
  <UiButton40 variant="secondaryBlue">Secondary Blue</UiButton40>
  <UiButton40 variant="secondaryFilled">Secondary Filled</UiButton40>
</Canvas>

### Disabled

All buttons supports `isDisabled` property:

<Canvas>
  <UiButton40 isDisabled type="button" variant="primary">Button</UiButton40>
  <UiButton40 href="#" isDisabled type="link" variant="primary">Link</UiButton40>
  <UiButton40 isDisabled type="visual" variant="primary">Visual</UiButton40>
</Canvas>

### Frozen

Along with `isDisabled` option, the component supports `isFrozen` option. In that case, button will have enabled visual
style, but will not react on user interactions.

<Canvas>
  <UiButton40 isFrozen type="button" variant="primary">Button</UiButton40>
  <UiButton40 href="#" isFrozen type="link" variant="primary">Link</UiButton40>
  <UiButton40 isFrozen type="visual" variant="primary">Visual</UiButton40>
</Canvas>

### Icon

You can provide an icon component:

<Canvas>
  <UiButton40 icon={AddIcon} variant="primary">Button</UiButton40>
  <UiButton40 icon={AddIcon} variant="secondary">Button</UiButton40>
  <UiButton40 icon={AddIcon} variant="secondaryBlue">Button</UiButton40>
  <UiButton40 icon={AddIcon} variant="secondaryFilled">Button</UiButton40>
</Canvas>

We recommend use icons with size of `16x16`.

An icon component should support `className` property.

### Analytics

Also, the component support the `trackId` property. It's adds `data-track-id` property to component for analytics
purposes.

<Canvas>
  <UiButton40 trackId="my-track-id" type="button" variant="primary">Tracked</UiButton40>
</Canvas>
