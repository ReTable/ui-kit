# UiButton40

Buttons allow users to take actions, and make choices. Simple example:

<Canvas>
  <UiButton40 variant="primary">Click me!</UiButton40>
</Canvas>

## Types

The component supports four types: `button`, `a`, `div` and `link`. By default, the `button` type is used.

### Button

By default, the component renders generic button:

<Canvas>
  <UiButton40 variant="primary">Default</UiButton40>
</Canvas>

You can provide `as="button"` explicitly:

<Canvas>
  <UiButton40 as="button" variant="primary">Button</UiButton40>
</Canvas>

### Anchor

You can render `a` element with providing `as="button"`:

<Canvas>
  <UiButton40 as="a" href="#" target="_blank" variant="primary">Anchor</UiButton40>
</Canvas>

### Div

In rare cases, you can render `div` element with providing `as="div"`:

<Canvas>
  <UiButton40 as="div" variant="primary">Div</UiButton40>
</Canvas>

### Link

You can use `as="link"` and provide `component` property to use `react-router`'s `Link` component:

<Canvas>
  <UiButton40 as="link" component={Link} target="_blank" to="#" variant="primary">Link</UiButton40>
</Canvas>

It can be useful, when you want to have `div` element with button appearance. For example, to render it inside
another `button` element.

## Options

The component supports a few options.

### `isDisabled`

This property allows to disable button:

<Canvas>
  <UiButton40 as="button" isDisabled variant="primary">Button</UiButton40>
  <UiButton40 as="a" href="#" isDisabled variant="primary">Anchor</UiButton40>
  <UiButton40 as="div" isDisabled variant="primary">Div</UiButton40>
  <UiButton40 as="link" component={Link} isDisabled to="#" variant="primary">Link</UiButton40>
</Canvas>

### `isFrozen`

This property allows to disable button, but keep enabled look and feel:

<Canvas>
  <UiButton40 as="button" isFrozen variant="primary">Button</UiButton40>
  <UiButton40 as="a" href="#" isFrozen variant="primary">Anchor</UiButton40>
  <UiButton40 as="div" isFrozen variant="primary">Div</UiButton40>
  <UiButton40 as="link" component={Link} isFrozen to="#" variant="primary">Link</UiButton40>
</Canvas>

### `isDisabled` and `isFrozen`

The `isDisabled` styles has priority over `isFrozen` when they're used together.

<Canvas>
  <UiButton40 as="button" isDisabled isFrozen variant="primary">Button</UiButton40>
  <UiButton40 as="a" isDisabled isFrozen href="#" variant="primary">Anchor</UiButton40>
  <UiButton40 as="div" isDisabled isFrozen variant="primary">Div</UiButton40>
  <UiButton40 as="link" component={Link} isDisabled isFrozen to="#" variant="primary">Link</UiButton40>
</Canvas>

### `icon`

You can provide an icon component:

<Canvas>
  <UiButton40 as="button" icon={AddIcon} variant="primary">Button</UiButton40>
  <UiButton40 as="a" icon={AddIcon} href="#" target="_blank" variant="primary">Anchor</UiButton40>
  <UiButton40 as="div" icon={AddIcon} variant="primary">Div</UiButton40>
  <UiButton40 as="link" icon={AddIcon} component={Link} target="_blank" to="#" variant="primary">Link</UiButton40>
</Canvas>

We recommend use icons with size of `16x16`.

An icon component should support `className` property.

## Attributes

All component types supports relevant HTML attributes, excluding `aria-disabled` and `disabled`. This attributes are
controlled by the component itself.

Also `href` attribute is restricted `link` type.

For example, you can provide `data-*` attribute for analytics purposes:

<Canvas>
  <UiButton40 as="button" data-track-id="button" variant="primary">Button</UiButton40>
  <UiButton40 as="a" data-track-id="anchor" href="#" target="_blank" variant="primary">Anchor</UiButton40>
  <UiButton40 as="div" data-track-id="div" variant="primary">Div</UiButton40>
  <UiButton40 as="link" data-track-id="link" component={Link} target="_blank" to="#" variant="primary">Link</UiButton40>
</Canvas>
