# UiButton24

Buttons allow users to take actions, and make choices. Simple example:

<Canvas>
  <UiButton24 variant="primary">Click me!</UiButton24>
</Canvas>

## Types

The component supports four types: `button`, `a`, `div` and `link`. By default, the `button` type is used.

### Button

By default, the component renders generic button:

<Canvas>
  <UiButton24 variant="primary">Default</UiButton24>
</Canvas>

You can provide `as="button"` explicitly:

<Canvas>
  <UiButton24 as="button" variant="primary">Button</UiButton24>
</Canvas>

### Anchor

You can render `a` element with providing `as="button"`:

<Canvas>
  <UiButton24 as="a" href="#" target="_blank" variant="primary">Anchor</UiButton24>
</Canvas>

### Div

In rare cases, you can render `div` element with providing `as="div"`:

<Canvas>
  <UiButton24 as="div" variant="primary">Div</UiButton24>
</Canvas>

### Link

You can use `as="link"` and provide `component` property to use `react-router`'s `Link` component:

<Canvas>
  <UiButton24 as="link" component={Link} target="_blank" to="#" variant="primary">Link</UiButton24>
</Canvas>

It can be useful, when you want to have `div` element with button appearance. For example, to render it inside
another `button` element.

## Options

The component supports a few options.

### `isDisabled`

This property allows to disable button:

<Canvas>
  <UiButton24 as="button" isDisabled variant="primary">Button</UiButton24>
  <UiButton24 as="a" href="#" isDisabled variant="primary">Anchor</UiButton24>
  <UiButton24 as="div" isDisabled variant="primary">Div</UiButton24>
  <UiButton24 as="link" component={Link} isDisabled to="#" variant="primary">Link</UiButton24>
</Canvas>

### `isFrozen`

This property allows to disable button, but keep enabled look and feel:

<Canvas>
  <UiButton24 as="button" isFrozen variant="primary">Button</UiButton24>
  <UiButton24 as="a" href="#" isFrozen variant="primary">Anchor</UiButton24>
  <UiButton24 as="div" isFrozen variant="primary">Div</UiButton24>
  <UiButton24 as="link" component={Link} isFrozen to="#" variant="primary">Link</UiButton24>
</Canvas>

### `isDisabled` and `isFrozen`

The `isDisabled` styles has priority over `isFrozen` when they're used together.

<Canvas>
  <UiButton24 as="button" isDisabled isFrozen variant="primary">Button</UiButton24>
  <UiButton24 as="a" isDisabled isFrozen href="#" variant="primary">Anchor</UiButton24>
  <UiButton24 as="div" isDisabled isFrozen variant="primary">Div</UiButton24>
  <UiButton24 as="link" component={Link} isDisabled isFrozen to="#" variant="primary">Link</UiButton24>
</Canvas>

### `icon`

You can provide an icon component:

<Canvas>
  <UiButton24 as="button" icon={AddIcon} variant="primary">Button</UiButton24>
  <UiButton24 as="a" icon={AddIcon} href="#" target="_blank" variant="primary">Anchor</UiButton24>
  <UiButton24 as="div" icon={AddIcon} variant="primary">Div</UiButton24>
  <UiButton24 as="link" icon={AddIcon} component={Link} target="_blank" to="#" variant="primary">Link</UiButton24>
</Canvas>

We recommend use icons with size of `16x16`.

An icon component should support `className` property.

## Attributes

All component types supports relevant HTML attributes, excluding `aria-disabled` and `disabled`. These attributes are
controlled by the component itself.

Also `href` attribute is restricted `link` type.

For example, you can provide `data-*` attribute for analytics purposes:

<Canvas>
  <UiButton24 as="button" data-track-id="button" variant="primary">Button</UiButton24>
  <UiButton24 as="a" data-track-id="anchor" href="#" target="_blank" variant="primary">Anchor</UiButton24>
  <UiButton24 as="div" data-track-id="div" variant="primary">Div</UiButton24>
  <UiButton24 as="link" data-track-id="link" component={Link} target="_blank" to="#" variant="primary">Link</UiButton24>
</Canvas>
