# UiButton24

Buttons allow users to take actions, and make choices. Simple example:

<Canvas>
  <UiButton24 variant="primary">Clickable Button</UiButton24>
</Canvas>

## Demos

The component supports are many options.

### As

The components supports three modes: `button`, `link` and `visual`. By default, the `button` mode is used.

#### Button

By default, the component renders generic button:

<Canvas>
  <UiButton24 variant="primary">Default</UiButton24>
  <UiButton24 as="button"  variant="primary">Button</UiButton24>
</Canvas>

#### Link

You can use button as link providing `as="link"` property, and additionally `href` and optionally `target`:

<Canvas>
  <UiButton24 as="link" href="#" variant="primary">Link</UiButton24>
  <UiButton24 as="link" href="#" target="_blank" variant="primary">Link with target</UiButton24>
</Canvas>

#### Visual

You can render button as `div` element providing `as="visual"` for rare cases when you can't use `button` element:

<Canvas>
  <UiButton24 as="visual" variant="primary">Visual</UiButton24>
</Canvas>

### Variant

There are 7 variants:

<Canvas>
  <UiButton24 variant="primary">Primary</UiButton24>
  <UiButton24 variant="secondary">Secondary</UiButton24>
  <UiButton24 variant="cancel">Cancel</UiButton24>
  <UiButton24 variant="cancelFilled">Cancel Filled</UiButton24>
  <UiButton24 variant="edit">Edit</UiButton24>
  <UiButton24 variant="test">Test</UiButton24>
  <UiButton24 variant="ai">Ai</UiButton24>
</Canvas>

### Disabled

All buttons supports `isDisabled` property:

<Canvas>
  <UiButton24 as="button" isDisabled variant="primary">Button</UiButton24>
  <UiButton24 as="link" isDisabled href="#" variant="primary">Link</UiButton24>
  <UiButton24 as="visual" isDisabled variant="primary">Visual</UiButton24>
</Canvas>

### Frozen

Along with `isDisabled` option, the component supports `isFrozen` option. In that case, button will have enabled visual
style, but will not react on user interactions.

<Canvas>
  <UiButton24 as="button" isFrozen variant="primary">Button</UiButton24>
  <UiButton24 as="link" isFrozen href="#" variant="primary">Link</UiButton24>
  <UiButton24 as="visual" isFrozen variant="primary">Visual</UiButton24>
</Canvas>

### Icon

You can provide an icon component:

<Canvas>
  <UiButton24 icon={AddIcon} variant="primary">Button</UiButton24>
  <UiButton24 icon={AddIcon} variant="secondary">Button</UiButton24>
  <UiButton24 icon={AddIcon} variant="cancel">Button</UiButton24>
  <UiButton24 icon={AddIcon} variant="cancelFilled">Cancel Button</UiButton24>
  <UiButton24 icon={AddIcon} variant="edit">Button</UiButton24>
  <UiButton24 icon={AddIcon} variant="test">Button</UiButton24>
  <UiButton24 icon={AddIcon} variant="ai">Button</UiButton24>
</Canvas>

We recommend use icons with size of `16x16`.

An icon component should supports `className` property.

### Analytics

Also, the component supports the `trackId` property. It's adds `data-track-id` property to component for analytics
purposes.

<Canvas>
  <UiButton24 as="button" trackId="my-track-id" variant="primary">Tracked</UiButton24>
</Canvas>
