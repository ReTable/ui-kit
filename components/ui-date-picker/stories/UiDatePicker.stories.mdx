# UiDatePicker

The `UiDatePicker` allows to select date, time or both date and time depends on selected type.

## Date

If you provide the `date` value to the `type` property, then only date selection will be available.

<Canvas>
  <DateStory.render />
</Canvas>

## Time

If you provide the `time` value to the `type` property, then only time selection will be available.

<Canvas>
  <TimeStory.render />
</Canvas>

## Date & Time

If you provide the `datetime` value to the `type` property, then date and time selection will be available.

  <DateTimeStory.render />
</Canvas>
